GIVEN an advisor has created a meeting note from a Contact (Only a Contact in the ‘Name’ field, ‘Related To’ will be Blank)
WHEN the advisor generates follow up actions for that meeting note
THEN we should only generate Tasks and we should NOT generate any Opportunity
•	GIVEN an advisor has created a meeting note from a Lead (only a Lead in the ‘Name’ field, ‘Related To’ will be Blank)
WHEN the advisor generates follow up actions for that meeting note
THEN we should only generate Tasks and we should not generate any Opportunity
